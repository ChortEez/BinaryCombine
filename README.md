## ДІЇ
- Множення 
  Має множити з проміжним доданком, виводити всі дії на екран. Також задля точності спочатку бере два множника та з двійкового переводить в десятковий, після чого множить та в кінці звіряється з результатом.

  ## Що зробити
  Перейди на бітсет, інакше при множені чи додавані покроково, це перетвориться просто на пекло для тебе
